Q2. The styleAll function assembles a style object containing Leaflet path options. It starts with default symbol properties, then checks each feature: if the feature is a point it enables stroke and radius, and if a postal code exists it swaps the fill to cyan; the completed style object is returned for Leaflet to render the marker.

Q4. In this context feature is the raw GeoJSON feature with its geometry and attribute properties, while layer is the Leaflet layer that was created to draw that feature on the map.

Q5. layer.options.fill is an attribute stored on the layer that controls how the symbol should be drawn, whereas layer.getLatLng() is a method call that calculates and returns a new Leaflet LatLng object describing the feature’s coordinates.
